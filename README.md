# 🚇 대구도시철도 2호선 도착 정보 조회 시스템  
**Daegu Subway Line 2 Arrival Information Viewer (C++ & libcurl)**

이 프로젝트는 **대구도시철도 2호선 열차 도착 시간을 실시간으로 계산하여 출력하는 C++ 프로그램**입니다.  
공공데이터포털(OpenData)에서 제공하는 **상행(영남대 → 문양) 열차 시각표 API**를 활용하여 특정 역(두류역)의 **다음 열차 도착까지 남은 시간**을 계산합니다.

---

## 📌 주요 기능

### ✔ 1. 공공데이터 API 연동
- `libcurl`을 이용해 OpenAPI JSON 자동 요청
- 서비스 키 URL 인코딩 처리
- 테스트 환경용 SSL 인증 무시 옵션 제공

---

### ✔ 2. JSON 데이터 파싱
- `nlohmann/json` 라이브러리 사용  
- 필요 필드만 추출하여 열차 시각 분석  
- **평일 / 토요일 / 공휴일 자동 판단**

---

### ✔ 3. 실시간 도착 정보 계산
프로그램 실행 시 현재 시간을 읽어:

- 이미 지나간 열차 제외  
- 남아 있는 열차들을 오름차순 정렬  
- **“몇 분 후 도착하는지” 자동 계산**

#### 예시 출력:
오늘 요일: 평일(상)
두류역 다음 열차 3개 (몇 분 후 도착):

4분 후

5분 후

12분 후


---

### ✔ 4. 상행(상선) 기준 계산
현재 OpenAPI는 **2호선 상행(영남대 → 문양)** 시간표만 제공함.  
따라서 본 프로그램은 **두류역에 문양 방향으로 들어오는 열차**의 도착 정보를 보여줍니다.
